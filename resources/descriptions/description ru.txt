modern scroll поднимает возможности прокрутки страницы на совершенно новый уровень.

Чем же он так хорош?

+ предоставляет браузеру всю ширину экрана для отображения веб-страниц, т.е. полосы прокрутки не занимают лишнего места
+ вся страница или её часть не дёргаются при смене содержимого когда полосы прокрутки становятся (не)нужными
+ полосы прокрутки также доступны в полноэкранном режиме
+ полосы прокрутки отображаются только когда они действительно нужны
+ внешний вид и поведение полностью настраиваемы
+ повышает эффективность при прокрутке страниц
+ позволяет регулировать скорость прокрутки
+ уникальная "супер панель" позволяет одновременно прокручивать по вертикали и по горизонтали
+ доступны кнопки для быстрого пролистывания к верхней/нижней части страницы
+ JavaScript интерфейс для создания пользовательских жестов мышкой и закладок
